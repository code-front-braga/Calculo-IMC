    O Índice de Massa Corporal (IMC) é uma ferramenta utilizada para avaliar se o peso de uma pessoa está dentro do valor ideal para a altura. Ele é calculado dividindo o peso (em quilogramas) pela altura ao quadrado (em metros), de acordo com a seguinte fórmula: IMC = PESO / ALTURA²

    O resultado do IMC é dado em kg/m². Aqui estão os passos para calcular o IMC: Peso: Converta o peso para quilogramas, se necessário. Altura: Converta a altura para metros, se necessário.
